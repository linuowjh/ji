## 🔧 可选配置

### 1. 配置后端API（可选）

编辑 `utils/api.js`：
```javascript
const BASE_URL = 'http://localhost:8080/api/v1'  // 改为你的后端地址
```

### 2. 配置AppID（生产环境）

编辑 `project.config.json`：
```json
{
  "appid": "your-real-appid"  // 替换为正式AppID
}
```

### 3. 替换图标（可选）

当前使用占位图标，可以替换为精美设计：
- 保持81x81尺寸
- 使用PNG格式
- 放在 `images/` 目录
